# Asta AI - توليد الصور بالذكاء الاصطناعي

تطبيق ويب لتوليد الصور باستخدام الذكاء الاصطناعي مع نظام نقاط وإعلانات.

## المميزات

- توليد صور عالية الجودة باستخدام الذكاء الاصطناعي
- نظام نقاط (كل صورة تكلف نقطة واحدة)
- إمكانية الحصول على نقاط مجانية من خلال مشاهدة الإعلانات
- معرض للصور السابقة
- خيارات أنماط متعددة (واقعي، كرتوني، خيالي، أنمي)
- مشاركة الصور على وسائل التواصل الاجتماعي
- تصميم متجاوب يعمل على جميع الأجهزة

## متطلبات النشر على Vercel

1. حساب على [Vercel](https://vercel.com)
2. مستودع Git (GitHub, GitLab, Bitbucket)
3. المتصفح الحديث

## خطوات النشر على Vercel

### الطريقة 1: النشر عبر واجهة Vercel

1. قم بإنشاء مستودع على GitHub وأرفع الملفات
2. سجل الدخول إلى [vercel.com](https://vercel.com)
3. انقر على "New Project"
4. اختر المستودع الخاص بك
5. اترك الإعدادات الافتراضية وانقر على "Deploy"
6. ستحصل على رابط مثل: `https://asta-ai.vercel.app`

### الطريقة 2: النشر عبر سطر الأوامر

```bash
# تثبيت Vercel CLI
npm install -g vercel

# تسجيل الدخول
vercel login

# النشر
vercel --prod
